सम्पोषि अध्यारोपण की स्थिति

Path Difference ZERO

  • s1 और s2 same phase के waves को emit कर रहा है |
  • ये दोनों same distance को cover करते हुए दिए गए points पर पहुँचता है |
  • इसलिए वहां same phase में पहुंचेगा,
  • और वहां constructive interference होगा,
  • और वहां Bright point होगा |

Path differeince λ

  • s1 और s2 same phase के waves को emit कर रहा है |
  • ये दोनों unequal distance को cover करते हुए दिए गए points पर पहुँचता है S1 से point की दुरी (PS1) ज्यादा और S2 से point की दुरी (PS2) कम है|
  • दोनों wave का path difference
  • Δx =PS1 – PS2= λ है | चूँकि path difference λ है इसलिए वहां दोनों wave same phase में पहुंचेगा,
  • और वहां constructive interference होगा, और वहां Bright point होगा |
